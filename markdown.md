# Markdown - `.md`

Invented in 2004, **Mark-_down_** is a [mark-_up_ language](https://en.wikipedia.org/wiki/Markup_language), used to create [hypertext](https://en.wikipedia.org/wiki/Hypertext) [documents](https://en.wikipedia.org/wiki/Writing). It is used accross [the web](https://en.wikipedia.org/wiki/World_Wide_Web), by **devs** and **users** alike.

## Background

- Web developers know and love Markdown. They use it to write software documentation like README and CONTRIBUTING files or API overviews.
- A lot of content centric sites, like **blogs** and technical **docs**, are build from markdown files.
- Posts and comments in some **apps** are Markdown(-based)

  - GitHub
  - Reddit
  - Stack Exchange (Stack Overflow)
  - Discord

## Elements

You can markup words, phrases or paragraphs within a Markdown document using _any combination_ of the following elements.

### `_Italic_`

Surround some text with `_` underscores. `This is _emphasized text_` becomes "This is _emphasized text_".

### `**Bold**`

Surround some text with `**` two asterisks. `This is a highlighted **keyword**` becomes "This is a highlighted **keyword**".
